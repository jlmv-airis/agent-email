Infraestructura: HostGator vs Google Cloud

Análisis de Despliegue
MÁS FÁCIL

HostGator (VPS n8n)

  • ✓ Instalación 1-Click: Ya viene configurado con el entorno n8n. No requiere scripts complejos.
  • ✓ Administración: Panel simplificado. Ideal para despliegue rápido.
  • ✓ Costo: Plan fijo (aprox. $15-$25 USD). Económico para presupuestos ajustados.
  • ⚠ Limitación: Menos integrado con el ecosistema nativo de Google (latencia).
MÁS ROBUSTO

Google Cloud (GCP)

  • ✓ Seguridad Nativa: Los datos de @airis-ae.com.mx nunca salen de la red de Google.
  • ✓ Escalabilidad: Potencia ilimitada. n8n corre en Docker con alta disponibilidad.
  • ✕ Configuración: Requiere montaje manual (Instancia, Firewall, Docker, SSL).
  • ✕ Administración: Curva de aprendizaje técnica alta para el área de IT.

Desarrollo: Mac M4 vs Dell i7

Conclusión: El equipo Dell actual tiene un procesador de bajo consumo que se satura al emular 50 hilos de ejecución de IA. La MacBook Air M4 (16GB) es necesaria para desarrollar el panel interactivo y los flujos n8n sin bloqueos técnicos, asegurando el plazo de 4 meses.

Requisitos Exactos de Google

🔑

Google Workspace Admin

Habilitar "Domain-Wide Delegation" para que la app gestione los 50 buzones de forma centralizada.

📡

Google Cloud Console

Activar Gmail API, Drive API y Admin SDK. Crear Credenciales OAuth 2.0 (Client ID / Secret).

🌐

Subdominio Dedicado

Ej: agent.airis-ae.com.mx para recibir Webhooks y alojar el Panel de Operadores.

Roadmap de Implementación: Asesorías IA

Mes 1: Cimientos

Configuración de HostGator/GCP, permisos de Google y primer despliegue de n8n.

Mes 2: Inteligencia IA

Programación de Triage automático, extracción OCR de INE/RFC y lógica de Drive.

Mes 3: Paneles Web

Desarrollo del Dashboard Admin y la interfaz de respuesta unificada para Operadores.

Mes 4: QA & Launch

Pruebas finales, capacitación de personal y margen para imprevistos técnicos.
