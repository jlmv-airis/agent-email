## 🎯 Objetivos Cumplidos

### 💻 Adaptación macOS (DevOps)
- [x] **Scripts Nativos:** Creados `INICIAR-SISTEMA.sh`, `DETENER-SISTEMA.sh` y `DIAGNOSTICO.sh` con permisos de ejecución.
- [x] **Gestión de Puertos:** Migración al puerto `8001` para evitar conflictos con AirPlay en macOS.

- [x] **IMAP Append:** Implementada la subida automática de correos respondidos a la carpeta "Enviados" del servidor (Webmail).
- [x] **Registro Local:** Inserción automática de mensajes enviados en la tabla `hilos` con `folder='SENT'`.
- [x] **Mapeo Inteligente:** Detección de nombres de carpetas (`Sent`, `Enviados`, `Borradores`, `Papelera`) según el proveedor de correo.
- [x] **Base de Datos:** Corrección de sintaxis en índices de SQLite y optimización de rendimiento.

### 🖥️ Interfaz de Usuario (Frontend)
- [x] **Diseño de Filtros:** Reestructuración de la barra de filtros en dos filas para evitar desbordamientos.
- [x] **Simplificación de Textos:** Filtros ahora usan etiquetas minimalistas: `TODOS`, `TODAS`, `CUALQUIERA`.
- [x] **Unificación de Estados:** Visualización de estados gestionados mediante cápsulas verdes `RESPONDIDO` en mayúsculas.
- [x] **Login Pro:** Añadido aviso visual de `BLOQUEO DE MAYÚSCULAS ACTIVADO` en el campo de contraseña.
- [x] **Footer:** Actualizada la versión global a `v1.0.11-mac`.
